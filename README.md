# mini-memo

## 共有リンク仕様
- 公開範囲は「非公開 / リンクを知っている人のみ」を選択できます。
- 共有URLはメモ本文ベースのハッシュを含むトークンで生成されます。
- 「共有停止」または「再生成」を実行すると旧リンクは即時無効化されます。
- 共有ページはタイトル・更新日時・本文のみを表示する read-only ビューです。
- 公開時にメール/電話/住所パターンを検知し、注意喚起を表示します。
- 共有メタ情報はメモ本体とは別の保存レイヤーで管理します。

## リマインダー仕様
- メモは `reminderAt`（nullable epoch ms）と `reminderDone` を保持します。
- エディタから日時指定・解除・通知済み化が可能です。
- 通知権限がない場合はアプリ内モーダル表示に自動フォールバックします。
- 一覧フィルタに「期限切れ / 今日 / 今週」を追加しています。
- JSON import/export はリマインダー情報を保持し、旧フォーマット（項目なし）も読み込み可能です。
