# mini-memo

## 概要
mini-memo は、ブラウザ上で動作するシングルファイル構成のメモアプリです。  
複数メモの管理に加えて、タグ・ノートブック・履歴・共有リンク・リマインダーなど、日常的なメモ運用に必要な機能を一通り備えています。

## 主な機能
- **複数メモ管理**: 新規作成、編集、検索、ピン留め、削除、復元に対応。
- **ノート整理**: タグ付けとノートブック分けでメモを分類可能。
- **履歴管理**: メモの保存履歴を表示し、過去バージョンとの比較・復元が可能。
- **ゴミ箱機能**: 削除メモを一時保管し、必要に応じて復元できる。
- **共有リンク**: メモごとに公開範囲を設定し、共有URLの生成・再生成・無効化ができる。
- **リマインダー**: 日時指定の通知、通知済み管理、期限ベースのフィルタに対応。
- **エディタ切り替え**: 編集ビューとプレビュービューを切り替えて確認可能。
- **インポート/エクスポート**: JSON形式でデータのバックアップと復元が可能（旧フォーマットも一部互換）。

## 共有リンク仕様
- 公開範囲は「非公開 / リンクを知っている人のみ」を選択できます。
- 共有URLはメモ本文ベースのハッシュを含むトークンで生成されます。
- 「共有停止」または「再生成」を実行すると旧リンクは即時無効化されます。
- 共有ページはタイトル・更新日時・本文のみを表示する read-only ビューです。
- 公開時にメール/電話/住所パターンを検知し、注意喚起を表示します。
- 共有メタ情報はメモ本体とは別の保存レイヤーで管理します。

## リマインダー仕様
- メモは `reminderAt`（nullable epoch ms）と `reminderDone` を保持します。
- エディタから日時指定・解除・通知済み化が可能です。
- 通知権限がない場合はアプリ内モーダル表示に自動フォールバックします。
- 一覧フィルタに「期限切れ / 今日 / 今週」を追加しています。
- JSON import/export はリマインダー情報を保持し、旧フォーマット（項目なし）も読み込み可能です。
